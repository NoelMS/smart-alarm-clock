SETUP INSTRUCTIONS FOR WINDOWS
==============================

1. Create folder: C:\Users\noelm\opencode\smart-alarm-clock\

2. Copy ALL FILES from this folder to that location

3. Open Arduino IDE and open: smart_alarm_clock.ino

4. Ensure these are installed in Arduino IDE:
   - ESP32 board support (via Board Manager)
   - 6 required libraries (via Library Manager)

5. Click Sketch → Verify to compile

6. Click Sketch → Upload to send to ESP32

7. When upload complete, open Tools → Serial Monitor

8. Later: Use Tools → ESP32 Sketch Data Upload for web files
